# haico Landing Page

**Ein KI-gestütztes Kundensimulations-Tool von philoneos**

## 📋 Projektübersicht

haico ist eine innovative Plattform, die Kundensegmentierung in gesprächsfähige digitale Personas verwandelt. Das Tool ermöglicht es Unternehmen, jederzeit mit KI-gestützten Kundenpersönlichkeiten zu kommunizieren, um Entscheidungen zu treffen und Feedback einzuholen.

### 🎯 Hauptfunktionen

- **KI-gestützte Kundensimulation**: Digitale Personas, die du jederzeit befragen kannst
- **Forschungsbasiert**: Akademisch fundiert und valide Methoden
- **DSGVO-konform**: Made in Germany mit Datenschutzgarantie
- **Direkter Zugriff**: Kundenperspektive ohne Terminabstimmung
- **24/7 Verfügbarkeit**: Immer ein Platz am Meetingtisch

## 🏗️ Technische Details

### Struktur
```
haico-landing-page/
├── index.html          # Haupt-Landingpage (Deutsch)
├── assets/             # Medien-Assets
│   ├── hero-video.mp4  # Hero-Video für den Header
│   ├── hero-poster.jpg # Poster-Image für das Video
│   └── og-image.jpg    # Open Graph Social Media Image
└── README.md           # Diese Datei
```

### Technologien
- **HTML5** mit semantischer Struktur
- **CSS3** mit modernen Features (Grid, Flexbox, Custom Properties)
- **Vanilla JavaScript** für Interaktivität
- **Responsive Design** für Mobile-First-Ansatz
- **SEO-optimiert** mit Meta-Tags und Open Graph
